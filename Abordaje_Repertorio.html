Abordaje del repertorio

June 16, 2020 posted by Admin Nat

este módulo incluye una variedad de herramientas digitales diseñadas para facilitar la práctica del repertorio específico dirigido a estudiantes en niveles iniciales. Este módulo se basa en la información recopilada a través de entrevistas con docentes y las fichas RAE.

A continuación. se presenta la herramienta pertinente: Vocal Remover: Es una página web gratuita que permite dividir la voz y la parte instrumental de una pista musical dejando como resultado dos pistas, la voz sin acompañamiento instrumental y una pista tipo karaoke. También posee más herramientas útiles para transportar la tonalidad de una canción, saber la tonalidad y la velocidad de esta, grabar voz, y grabar sobre una pista tipo karaoke. Es una herramienta muy completa. A continuación, se prensa el enlace correspondiente a la herramienta: Vocal Remover Web
